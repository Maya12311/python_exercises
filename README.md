# Python Exercises

## Overview

This repository contains solutions to different Python exercises focusing on object-oriented programming concepts.

## Exercise: Encapsulation

### Description

In this exercise, an `Athlete` class is implemented using encapsulation principles.

### Features

* Private attributes (`runner_name`, `gender`)
* Getter and setter methods
* Conditional behavior based on gender

### Example

```python
maria = Athlete("Maria", "girl")
print(maria.running())
```

### Output

```
150mtr running
```
